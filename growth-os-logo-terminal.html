// Implementation Notes
Font: JetBrains Mono — weight 700 for wordmark, 400 for supporting text
Wordmark: growth_os — "growth" in #f0f0f0, "_os" in #10b981
The underscore _ animates with a blink (1.1s step-end) on landing page hero only — static everywhere else
App icon: g_o in dark rounded square with emerald border at 0.35 opacity
Favicon (16px): solid emerald square with go in black — simplified for tiny size
Never use the logo on a coloured background other than #0a0a0a, #111111, or white
Install font: npm install @fontsource/jetbrains-mono then import in layout.tsx
